# Demo for 天气预报查询机器人


## 功能
这个机器人可以根据你提供的城市（北京、上海等）和日期（明天、后天等），查询出相应的天气预报。

## 特性
使用 Frame-based 对话管理方案，如果上述两个 Slot (既城市和天气)，有任意一个用户未提供，对话管理系统会负责让你澄清相关 Slot 的值。

## 在线演示
见页面右下方的聊天widget ![chat_button](chat_button.png)，点击即可使用
